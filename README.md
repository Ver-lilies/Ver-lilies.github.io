# 多效果实时哈哈镜

这是一个课程设计项目，用于通过摄像头实时采集图像，并使用 OpenCV / OpenCV.js 实现多种哈哈镜扭曲效果。

## 功能

- 通过电脑或手机摄像头实时采集图像。
- 提供正常、波浪、凸透镜、凹透镜、旋涡 5 种显示模式。
- 使用图像坐标重映射算法处理摄像头画面。
- 支持调节扭曲程度，范围为 `0` 到 `1.5`。
- Web 端支持拍照保存，并可关闭/重新打开摄像头。
- Python 端支持键盘切换效果、调节强度和拍照保存。

## Web 版运行

直接访问 GitHub Pages 页面，或在本地启动静态服务：

```bash
python3 -m http.server 8000
```

然后在浏览器打开：

```text
http://127.0.0.1:8000/index.html
```

首次使用需要允许浏览器访问摄像头。手机浏览器建议使用 HTTPS 页面访问，摄像头权限更稳定。

## Python 版运行

安装依赖：

```bash
pip install -r requirements.txt
```

运行程序：

```bash
python main.py
```

键盘操作：

- `0`：正常画面
- `1`：波浪扭曲
- `2`：凸透镜效果
- `3`：凹透镜效果
- `4`：旋涡扭曲
- `a` / `d`：减小 / 增大扭曲程度
- `s`：拍照保存
- `q`：退出程序

如果外接 USB 摄像头不是默认设备，可以在 `main.py` 中将 `cv2.VideoCapture(0)` 的编号改为 `1` 或 `2`。
